## Example data for the ALTRE R package

A restricted subset of the data with one chromosome (i.e. chromosome 21) is here. The corresponding CSV file for input into ALTRE is included.


To download the entire data, please use a *file download manager* to download the files from the links listed below. 

### Alignment (in BAM format) files:

#### *A549*:
